# OpenDyslexic Fonts & Resources

 For the OpenDyslexic font.


- OpenDyslexic (Newest) / 2025

OpenDyslexic: a typeface for dyslexia.

OpenDyslexic is a typeface designed to alleviate some common symptoms of dyslexia. 

OpenDyslexic is free: its latest version uses the SIL-OFL license, allowing you to use it freely for personal, professional, educational, commercial purposes, for books, e-readers, applications, websites, and for any other project.

Version: OpenDyslexic V. 29 November 2025
